This is a personal project to manage file and folder from remote storage.

## Getting Started

First, install all dependencies:
```bash
pnpm install
```

Then, set the environment variables `UPLOAD_PATH` to the directory where you would like to store your data like this:

```bash
export UPLOAD_PATH=<your directory>
```

Run these commands to build and start the website:

```bash
pnpm build
pnpm start
```

Open [http://[your_hostname]:3000]() to see and enjoy the result.
